# Space Race

A simple racing game, set in space.

## To install

```
$ bundle
```

## To play

```
$ ruby main.rb
```

Steer the ship with the left and right arrow keys. Dodge as many opponents as
you can before you run out of fuel! Catch fuel pellets to extend the game.
